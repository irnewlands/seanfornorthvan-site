# SeanForNorthVan.ca Website

Developer-friendly campaign site built with React + Vite.

## Quick Start
```bash
npm install
npm run dev
```

## Deploy to Cloudflare Pages
1. Push this folder to a GitHub repository.
2. In Cloudflare Pages, choose **Connect to Git**.
3. Select the GitHub repo.
4. Framework preset: **Vite**.
5. Build command: `npm run build`.
6. Build output directory: `dist`.

## Domain Setup
Use GoDaddy DNS or Cloudflare DNS. For Cloudflare Pages, add the custom domain `seanfornorthvan.ca` inside Cloudflare Pages and follow the DNS instructions.

## Brand Notes
Use navy blue, olive green, white, subtle gold, North Vancouver imagery, clean civic typography, strong icon-led layouts, and professional Sean photography.

## Content Notes
The platform language is based around: **Your Voice. Our City. No More Surprises.**
